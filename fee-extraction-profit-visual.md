```

═══════════════════════════════════════

┌────────────────────┐

│ ПРОФИТ: +$1.25k ✅ │
└────────────────────┘

Вы субсидируете пул для привлечения объема!
```

## ⚡ Критические факторы

```
ЧТО ДЕЛАЕТ СТРАТЕГИЮ ПРИБЫЛЬНОЙ:

1. РАЗМЕР LP ДОЛИ
   ├─ 5%  = Скорее всего убыток ❌
   ├─ 15% = Может быть профит ⚠️
   ├─ 25% = Хороший профит ✅
   └─ 40% = Отличный профит ✅✅

2. ОРГАНИЧЕСКИЙ ОБЪЕМ
   ├─ $0 = Гарантированный убыток ❌
   ├─ = Вашему FL = Безубыточность
   └─ > 2x вашего FL = Профит! ✅

3. КОМИССИЯ ПУЛА
   ├─ 0.05% = Слишком мало ❌
   ├─ 0.25% = Норма ✅
   └─ 1%    = Отлично! ✅✅
```
