# Визуальная схема процесса миграции на LetsBONK.fun

## Полный цикл жизни токена

```
┌─────────────────────────────────────────────────────────────────┐
│                      СОЗДАНИЕ ТОКЕНА                             │
├─────────────────────────────────────────────────────────────────┤
│  1. Пользователь создает токен на LetsBONK.fun                  │
│  2. Устанавливается начальная bonding curve                     │
│  3. Токен становится доступен для торговли                      │
└──────────────────────────┬──────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────────────┐
│                    ФАЗА BONDING CURVE                            │
├─────────────────────────────────────────────────────────────────┤
│  • Торговля через автоматизированную кривую                     │
│  • Цена растет с каждой покупкой                               │
│  • Цена падает с каждой продажей                               │
│  • Вся ликвидность хранится в контракте                        │
│                                                                  │
│  Целевые показатели для graduation:                             │
│  ✓ Market Cap: $69,000+                                         │
│  ✓ Holders: 100+                                                │
│  ✓ Volume: Достаточный для устойчивости                         │
└──────────────────────────┬──────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────────────┐
│                    ПРОВЕРКА УСЛОВИЙ                              │
├─────────────────────────────────────────────────────────────────┤
│  if (marketCap >= $69,000 &&                                    │
│      holders >= 100 &&                                          │
│      volume24h >= threshold) {                                  │
│      → ИНИЦИИРОВАТЬ МИГРАЦИЮ                                    │
│  }                                                              │
└──────────────────────────┬──────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────────────┐
│                  ПРОЦЕСС МИГРАЦИИ (GRADUATION)                   │
├─────────────────────────────────────────────────────────────────┤
│  1. Блокировка торговли на bonding curve                        │
│  2. Расчет финальной цены                                       │
│  3. Подготовка ликвидности для переноса                         │
└──────────────────────────┬──────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────────────┐
│                   СОЗДАНИЕ ПУЛА НА RAYDIUM                       │
├─────────────────────────────────────────────────────────────────┤
│  1. Автоматическое создание пула Raydium                        │
│  2. Перенос всей ликвидности из bonding curve                   │
│  3. Установка начальной цены = финальной цене на curve          │
│  4. Активация торговли на Raydium                               │
└──────────────────────────┬──────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────────────┐
│                    ИНТЕГРАЦИЯ С JUPITER                          │
├─────────────────────────────────────────────────────────────────┤
│  • Автоматическое добавление в маршрутизатор Jupiter            │
│  • Доступность через все DEX агрегаторы                         │
│  • Максимальная ликвидность и видимость                         │
└──────────────────────────┬──────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────────────┐
│                    ПОСТМИГРАЦИОННАЯ ФАЗА                         │
├─────────────────────────────────────────────────────────────────┤
│  • Торговля на Raydium и других DEX                             │
│  • Комиссии: 0.1-0.2% за транзакцию                            │
│  • Часть комиссий идет на выкуп и сжигание BONK                │
│  • Полная децентрализация торговли                              │
└─────────────────────────────────────────────────────────────────┘
```

## PumpSwap - Миграция между платформами

```
┌─────────────────┐                    ┌─────────────────┐
│   PUMP.FUN      │                    │  LETSBONK.FUN   │
│                 │                    │                 │
│  Токен на       │    🔄 PumpSwap    │  Токен на       │
│  Pump.fun       │ ================► │  LetsBONK.fun   │
│                 │                    │                 │
│                 │  Бесплатно с      │                 │
│                 │  марта 2025       │                 │
└─────────────────┘                    └─────────────────┘
```

## Ключевые преимущества автоматической миграции

1. **🚀 Скорость**: Мгновенная миграция при достижении целей
2. **💧 Ликвидность**: 100% ликвидности переносится на Raydium
3. **🔗 Интеграция**: Автоматическое подключение к Jupiter
4. **💰 Экономия**: Не нужно платить за создание пула вручную
5. **🛡️ Безопасность**: Аудированные смарт-контракты
6. **📈 Непрерывность**: Сохранение цены при миграции

## Риски и важные моменты

⚠️ **Волатильность**: Мем-токены крайне рискованны
⚠️ **Rug Pull**: Всегда проверяйте контракты
⚠️ **IL Risk**: Риски предоставления ликвидности
⚠️ **Комиссии**: Учитывайте все комиссии в расчетах
