## 🎯 Формула безубыточности

```
МИНИМАЛЬНАЯ LP ДОЛЯ = FL fee ÷ Pool fee

Примеры:
• FL 0.05%, Pool 0.25% = Нужно 20% минимум
• FL 0.05%, Pool 0.05% = Нужно 100% (невозможно!)
• FL 0.05%, Pool 1% = Достаточно 5%!

ЗОЛОТОЕ ПРАВИЛО:
Ваша доля × Комиссия пула > FL комиссия
```

## 🎮 Стратегии для разных долей

### Если у вас 1-5%:

```
ЕДИНСТВЕННЫЙ ПУТЬ:
• Мем-токены с 1% fee
• Новые пулы (мало конкурентов)
• Ждать высокую активность
• FL только если видите объем
```
