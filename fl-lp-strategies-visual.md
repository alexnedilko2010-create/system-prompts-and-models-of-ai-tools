## 🚀 Стратегия 1: JIT Liquidity (Just-In-Time)

### Как работает:

```
ОБЫЧНЫЙ ПУЛ:              JIT АТАКА:
═════════════════         ═════════════════════

LP провайдеры:            Вы видите в mempool:
• Alice: 20%              "Боб хочет свапнуть 1000 SOL"
• Bob: 30%                         ↓
• Carol: 50%              За миллисекунды до Боба:
                          1. FL: 10,000 SOL + USDC
Боб свапает →             2. Добавить в пул
Комиссии делятся          3. Стать 80% пула!
между всеми                        ↓
                          Боб свапает через ВАС
                          Вы получаете 80% комиссий!
                                   ↓
                          Сразу после: убрать LP
═════════════════         ═════════════════════

Профит: 1000 × 0.3% × 80% = 2.4 SOL за секунду!
```

## 🌊 Стратегия 4: Impermanent Loss Harvesting

### Использовать IL в свою пользу:

```
СЦЕНАРИЙ: Знаете что токен скоро взлетит

До пампа:                 Во время пампа:
┌───────────────┐        ┌─────────────────┐
│ SOL: $100     │        │ SOL: $200 🚀    │
│ Пул 50:50     │   →    │ IL происходит   │
│ Мало объема   │        │ НО! Огромный    │
│               │        │ объем = fees!   │
└───────────────┘        └─────────────────┘

ВАШ FL ПЛАН:
1. Добавить LP перед пампом (FL)
2. Собрать fees от FOMO объема
3. Убрать LP после пампа
4. Fees > IL = ПРОФИТ!
```
